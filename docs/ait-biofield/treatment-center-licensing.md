# AIT Treatment Center Licensing Overview

## Licensing Paths
- Retrofit package for existing dialysis/treatment centers
- Standalone center model for new facilities

## Package Elements
- Equipment scope and installation constraints
- Protocol access and usage rights
- Compliance gates and review checkpoints
- Site readiness scoring and attestations
- Revenue model linkage (subject to legal review)

## Proof and Governance
Each licensing package links to document hashes, diligence status, compliance state, and anchor references in the protocol registry.
