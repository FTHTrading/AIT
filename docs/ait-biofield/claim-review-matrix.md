# AIT Claim Review Matrix

## Classification Bands
- Company-provided claim
- Needs clinical validation
- Needs regulatory review
- Needs legal review
- Public-safe
- Restricted

## Review Workflow
1. Register claim with source reference
2. Assign claim type: medical/technical/financial/operational/regulatory
3. Define required evidence
4. Classify risk level
5. Set publication eligibility
6. Hash and anchor review packet

## Publication Rule
No medical or financial claim is published without source, classification, and review status.
