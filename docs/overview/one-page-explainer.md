# One-Page Explainer

AIT provides the source technology.
UnyKorn provides the infrastructure for proving, governing, licensing, and monetizing the surrounding digital objects.

Core capabilities:

- Document hashing and proof manifests
- Merkle batch creation and chain-ready anchor payloads
- Claim classification and gated review status
- RWA metadata packaging for documented rights
- x402 challenge, receipt, and metering scaffolds
- Durable admin review queues

This repository is infrastructure only. It is not medical advice, clinical validation, a securities offering, or a patient record system.
