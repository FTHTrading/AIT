# Glossary

## AIT
Autologous Infusion Technologies and Therapies.

## Proof Manifest
A typed record containing a SHA-256 hash, visibility level, storage recommendation, and related proof metadata.

## Merkle Batch
A deterministic grouping of document hashes into a root suitable for anchoring.

## Anchor Payload
An unsigned chain-specific payload for Polygon, XRPL, or UnyKorn L1 anchoring.

## Visibility Level
One of `public`, `restricted`, or `private`.

## Review Status
Compliance state for a claim, proof, or package, including `PENDING_REVIEW`, `READY_FOR_ANCHOR`, and `ARCHIVED`.

## x402
A paid-access challenge and receipt model used to meter protected routes.
